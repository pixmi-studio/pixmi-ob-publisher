# Initial Concept

我想做一个 obsidian 的插件, 主要用于将笔记发布到微信公众平台的草稿箱.我期望这个插件开源到 github,是一个文档完善的精品开源项目.且实际的功能也要符合直觉,体验优秀.

# Product Guide: Pixmi Obsidian WeChat Publisher

## Target Users
- **Content Creators & Bloggers:** Users who leverage Obsidian's powerful editing features to draft long-form content.
- **Knowledge Management Enthusiasts:** Individuals who want to easily share their distilled insights with their WeChat audience.
- **Developers:** Users seeking a seamless, local-first workflow that eliminates the friction of manual copying and pasting.

## Core Features
- **One-Click Publishing:** Effortlessly push the active Obsidian note directly to the WeChat Official Account draft box.
- **Automated Asset Management:** Automatically handles image uploads to the WeChat media library and replaces local paths with permanent WeChat URLs.
- **What You See Is What You Get (WYSIWYG) Styling:** Ensures the published WeChat article maintains a visual style that is highly consistent with the Obsidian preview mode.
- **Intuitive Configuration:** A streamlined setup process that gets users up and running with minimal technical overhead.

## User Experience (UX) Goals
- **Minimalist Setup:** Focus on "out of the box" functionality with sensible defaults.
- **Resilient Feedback:** Provide clear, actionable notifications and error messages during the publishing process to build user trust.
- **Frictionless Workflow:** Design the interface to feel like a native extension of the Obsidian environment, prioritizing speed and ease of use.

## Open Source & Community
- **High-Quality Standards:** Aim to be a "gold standard" open-source project on GitHub, showcasing clean code and professional maintenance.
- **Transparent & Trustworthy:** Maintain a clear and open codebase to ensure user data (API keys, content) is handled securely.
- **Automated Lifecycle:** Employs industry-standard CI/CD practices for automated testing and releases, ensuring stability and professional version management.
- **Comprehensive Documentation:** Provide a rich documentation suite including:
    - Detailed User Guides with visual aids (GIFs/Screenshots).
    - Technical documentation for potential contributors.
    - A transparent project roadmap and clear contribution guidelines.
- **Verified by Community:** Successfully submitted to the Obsidian Community Plugins directory to ensure accessibility and trust for the wider Obsidian community.
