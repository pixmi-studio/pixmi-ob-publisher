# Track Spec: Community Store Submission

## Overview
This track covers the process of submitting the "Pixmi Obsidian WeChat Publisher" plugin to the official Obsidian Community Plugins directory to make it discoverable and installable by all Obsidian users.

## Requirements
- Valid `manifest.json` in root.
- `README.md` with usage instructions.
- `LICENSE` file.
- GitHub Release with `main.js` and `manifest.json`.
- Entry in `obsidian-releases/community-plugins.json`.

## Success Criteria
- [ ] Pull Request submitted to `obsidianmd/obsidian-releases`.
- [ ] Plugin reviewed and approved by the Obsidian team.
- [ ] Plugin appears in the Obsidian community plugin browser.
