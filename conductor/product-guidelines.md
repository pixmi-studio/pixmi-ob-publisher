# Product Guidelines: Pixmi Obsidian WeChat Publisher

## Brand Personality
- **Professional, Modern & Welcoming:** The project should feel like a high-end, reliable tool that is simultaneously cutting-edge and easy to approach. It balances technical excellence with user-centric design.

## Visual Identity & UI Design
- **Obsidian-Native Integration:**
    - The plugin's interface must feel like a natural extension of Obsidian.
    - Full support for both light and dark themes, using Obsidian's built-in CSS variables.
    - Adhere to Obsidian's layout and component patterns for a consistent user experience.

## Communication Style
- **Prose Style:**
    - **Concise & Action-Oriented:** Documentation and UI text should prioritize clarity and directness, helping users achieve their goals efficiently.
    - **Inclusive & Welcoming:** Use friendly, gender-neutral language to foster a diverse and helpful community.
- **Error Handling & Feedback:**
    - **Informative & Solution-Oriented:** Errors should be clearly explained in non-technical terms when possible, providing the user with a specific action or "next step" to resolve the issue.

## Principles & Values
- **Privacy-First & Local-First:**
    - User content and sensitive API credentials (e.g., WeChat AppID, AppSecret) must be stored securely and locally within the Obsidian environment.
    - All communication with the WeChat Official Account platform must be direct from the user's client, with no intermediate servers processing or storing private data.
- **Transparency:** Maintain an open and honest relationship with the community regarding project development, security practices, and data handling.
