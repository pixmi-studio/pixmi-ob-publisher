# Plugin Submission: Pixmi Obsidian WeChat Publisher

## Description
This plugin allows users to publish their Markdown notes directly to their WeChat Official Account draft box. It handles image uploading and converts Markdown to a format compatible with WeChat's editor.

## Key Features
- Direct publishing to WeChat drafts.
- Automatic image processing and uploading.
- Support for API proxies to bypass IP whitelist restrictions.
